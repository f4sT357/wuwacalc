ğŸ¨ Basic Settings

You can freely customize the application's appearance from the "Display Settings" dialog.

  • Theme Settings:
    • Dark: Default dark mode. Easy on the eyes.
    • Light: Bright color scheme. Suitable for daytime use.
    • Clear: Transparency-focused theme. Great match for background images.
  • Background Image: You can set your favorite image (PNG, JPG, BMP) as the background. Select a file using the "Select Image" button.
    ğŸ’¡ Tip (Relative Path):
    If you place the image file in the same folder as the application (or a subfolder inside it) before selecting it, the setting is automatically saved as a relative path.
    This allows the background image setting to be maintained even if you move or copy the entire tool folder to another location.
  • Background Opacity: You can adjust the window transparency with the slider. Controls how much the background image shows through.

ğŸŒˆ Color Customization

More detailed color adjustments are possible.

  • Text Color: Depending on the background image, text might become hard to read. In that case, change the text color to improve visibility.
  • Input Background Color: You can individually change the background color of drop-down menus and text boxes. Pressing the "Reset" button reverts it to the theme's default color.
  • Font Settings: You can choose from compatible fonts. Selecting "System Default" returns to the original font.

ğŸ’¡ Setting Examples

Night Mode
Theme: Dark
Opacity: 90% - 95%
Input Background: #121212 (Ultra Dark Gray)
Effect: Settings that reduce eye strain even in very dark environments.
Glassmorphism
Theme: Clear
Background Image: Landscapes or abstract images
Opacity: 70% - 80%
Effect: The background shows through moderately, creating a futuristic and stylish look.
High Contrast
Theme: Dark
Text Color: #Yellow or #Cyan
Input Background: #000000 (Black)
Effect: Effective when you want to maximize visibility.

âš ï¸ Precautions

  • Visibility: If the background image is too complex or the text color is similar to the background color, text may become difficult to read. Adjust opacity and text color to ensure readability.
  • Image Size: Using images with overly high resolutions may slow down the application. We recommend images of appropriate size (around Full HD).
  • Configuration Errors: If color settings become strange and you cannot revert them, please press the "Reset All Display Settings" button. All settings will return to their initial state.
