WuwaCalc User Guide

アプリの概要

WuwaCalc は、鳴潮（Wuthering Waves）の音骸スコアを多角的に評価するためのツールです。 OCR（光学文字認識）による自動入力機能を備え、複数の評価指標から音骸の「真の価値」を算出します。

基本的な使い方

1 キャラクターの設定
「キャラ設定」ボタンから、育てたいキャラクターを登録します。各ステータスの重要度（重み）を設定することで、そのキャラにとっての最適な評価が可能になります。
2 音骸データの入力
  • OCR（自動）: ゲーム画面のスクリーンショットをクリップボードにコピー（Win+Shift+S 等）し、アプリで Ctrl+V を押すだけで自動入力されます。
  • 手動: 各スロットのステータスと数値を直接入力します。
3 スコア計算
F5 キーまたは Ctrl+Enter を押すと計算が実行されます。 「自動計算」にチェックを入れると、データ入力と同時に計算が行われます。

評価方式の解説

  • 正規化 (GameWith式): 各サブステータスを最大値で割り、重みを掛けた標準的な方式です。
  • 比率重視 (Keisan式): サブステータスの比率とダメージ上昇期待値を重視した評価です。
  • ロール品質: 各ステータスが「最大値に対してどの程度の数値で跳ねたか」を評価します。
  • 有効ステ数: 有効なステータスがいくつ付いているかを基準にします。
  • CV換算 (標準): 会心率と会心ダメージをベースにした、コミュニティで最も一般的な指標です。

便利な機能と設定

  • 表示設定: 背景画像、不透明度、テーマ（ダーク/ライト/クリア）を自由に変更できます。
  • 計算履歴: 過去に計算した音骸のデータを自動保存します。厳選の進捗確認に役立ちます。
  • 前処理設定: OCRのエンジン（標準/OpenCV）を切り替えられます。読み取り精度が低い場合に調整してください。

OCR精度向上のコツ

  • スクリーンショットは、音骸のステータス部分が隠れないように撮影してください。
  • 実数値と%の誤認防止: 攻撃力などが「.」を含み、かつ20未満の場合は自動的に「%」として扱う補正機能が働きます。
  • 文字化けが発生する場合は、「前処理設定」から OpenCV モードを試してみてください（別途 pip install opencv-python が必要です）。
