Overview

A tool to calculate and evaluate Echo scores for Wuthering Waves.

It supports automatic input via OCR (image recognition) and manual input. Character-specific weighting settings are also available.

Features

  • Score Calculation: Evaluates using 5 methods (Normalization, Ratio, Roll Quality, Effective Stats Count, Crit Value).
  • OCR Input: Automatically reads stats from screenshots or clipboard images.
  • Batch Calculation: Evaluates multiple registered Echoes at once.
  • Character Settings: Set and save effective sub-stats and weights for each character.
  • Customizable Appearance: Supports theme changes and custom background images. Placing images in the application folder maintains settings even if the folder is moved (relative path support).
    For details, please refer to the Appearance Customization Guide.

Weighting Settings Guide

This tool uses a common "Weight" setting for multiple calculation logics, but how weights are handled differs by method. Follow these recommendations to ensure accurate evaluations across all methods.

Recommended Settings

  • Essential Stats (e.g., Crit Rate/DMG): Set to 1.0.
  • Effective Stats (e.g., Atk%): Set to 0.5 or higher (e.g., 0.7 ~ 1.0).
    • âš ï¸ Note: In "Effective Stats Count" method, stats with weight less than 0.5 are NOT counted as "effective".
  • Bonus Stats: Set to 0.1 ~ 0.4.
    • Added to "Normalized Score" but NOT included in "Effective Stats Count".
  • Useless Stats: Set to 0.

How Each Calculation Method Uses Weights and Formulas

The 5 calculation methods use weight coefficients in different ways. Understanding each method's characteristics and formulas allows for more appropriate weight settings.

Note: In the formulas below, Level Scaling refers to (Current Echo Level / 25). The score will be reduced unless the level is 25.

Method 1: Normalized Score (GameWith)

Role of Weights: Directly reflects the value of substats.

  • Formula: (15 + Î£(Sub-stat Score)) * Level Scaling
    • Sub-stat Score = (Sub-stat Value / Sub-stat Max Value / 5) * Weight * 100
    • A fixed 15 points are given for the main stat, to which each sub-stat's score is added.
  • Characteristics: All substats contribute to the score according to their weights, enabling balanced evaluation.

Method 2: Ratio-focused Method (Keisan)

Role of Weights: Represents the importance of each stat in damage calculation.

  • Formula: Î£( (Sub-stat Value / Sub-stat Max Value / 5) * Weight ) * 100 * Level Scaling
    • This sums up the weighted ratio of each sub-stat relative to its maximum possible value.
  • Characteristics: Enables evaluation close to actual damage calculation, making fine-tuning based on character builds important.

Method 3: Roll Quality Method

Role of Weights: Represents the importance when evaluating the "roll" value of each stat.

  • Formula: (Î£(Quality Points * Weight) / (Number of Target Stats * 3)) * 100 * Level Scaling
    • Quality Points: Awarded based on the sub-stat's value: Max (3), Good (2), Low (1), or Below (0.5).
    • Target Stats: Crit Rate, Crit DMG, Energy Regen, and ATK%.
  • Characteristics: Emphasizes the "strength of rolls." High-quality rolls on high-weight stats significantly increase the score.

Method 4: Effective Stats Count

Role of Weights: Functions as a threshold (0.5) to determine if a stat is "effective".

  • Formula: Î£(Contribution) * Bonus Multiplier * Level Scaling
    • Contribution: (Sub-stat Value / Max Value) * Weight * 20 (only for stats with weight â‰¥ 0.5).
    • Bonus Multiplier: Varies with the number of effective stats (5: 1.2, 4: 1.1, 3: 1.0, 2: 0.8, 1: 0.6).
  • Characteristics: A 1.2x bonus is applied for 5 effective stats. Whether the weight is above or below 0.5 is more critical than its exact value.

Method 5: CV (Crit Value) Method (Community Standard)

Role of Weights: Used only for certain damage bonus stats to reflect character-specific scaling.

  • Formula: (Sum of various scores) * Level Scaling
    • Crit Score = (Crit Rate * 2) + Crit DMG
    • ATK Score = (ATK% * 1.1) + (Flat ATK / 10 * 1.2)
    • Other Score = (Energy Regen * 0.5) + (Misc. DMG Bonuses * 1.1 * Weight)
  • Characteristics: The most widely used community method, heavily valuing Crit stats.

âš ï¸ Important Notes

  • Common Weight Settings: This tool uses the same weight settings for all calculation methods. To achieve balanced evaluation across all methods, refer to the recommended settings above.
  • 0.5 Threshold: To be considered effective in the "Effective Stats Count" method, weights must be set to 0.5 or higher.
  • Character-Specific Adjustments: Adjusting weight settings according to character builds and playstyles enables more accurate evaluation.
